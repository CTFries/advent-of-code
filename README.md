# Advent of Code, 2022 in GO

As a JavaScript engineer, I am using AoC to learn GO in 2022.

I will be leveraging the excellent resources of Front End Masters - [Go for JS Devs](https://frontendmasters.com/courses/go-for-js-devs/)

I also borrowed the project structure and build script from the excellent [Bozdoz](https://github.com/bozdoz/advent-of-code-2022/tree/b9de891e852ea754bbe1ec0278075acc082e5897)

## How to utilize this repo

- navigated to `/2022/go/`
- initialize a day: `./build-day.sh 01`
- copy puzzle input to `01/input.txt`
- copy puzzle text to `01/README.md`
- modify `01/main.go` to solve the puzzle
- build utilities in `/utils/` as needed to solve the puzzle & maintain DRY principles
- run `go run 01/main.go` to see the output of your solutions
